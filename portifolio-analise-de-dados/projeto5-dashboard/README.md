# Dashboard no Looker Studio

## Sobre o Projeto
Criação de um dashboard interativo no Looker Studio para visualização de indicadores e análise de dados.

## Ferramenta Utilizada
- **Looker Studio** (Google) - gratuito e integrado com Google Sheets/Excel

## O que o Dashboard Mostra
- [Descreva os gráficos que você colocou]
- Exemplo: distribuição de salários
- Exemplo: quantidade por nível de ensino
- Exemplo: análise por gênero

## Como Fiz
1. Fonte de dados: planilha do Excel/Google Sheets
2. Conexão direta com o Looker Studio
3. Criação de gráficos (barras, pizza, linha)
4. Adição de filtros interativos
5. Organização visual para storytelling
